# MVP Scope

## 목표

MVP는 현실적인 고품질 도시 GLB가 아니라 파이프라인 계약 무결성을 증명한다.

## 포함

- Phase 0 Foundation Docs
- Phase 1 Schema Contracts
- Phase 2 Fixtures First
- Phase 3 Provider Snapshot MVP
- Phase 4 Graph and Intent MVP
- Phase 5 Minimal MeshPlan and GLB

## 제외

- 포토리얼 facade
- roof equipment 자동 생성
- signage detail
- city-scale streaming
- unresolved conflict 자동 보정
- weather/traffic 기반 geometry 변경
