- `PROVIDER_*`
- `COMPLIANCE_*`
- `SPATIAL_*`
- `SCENE_*`
- `GEOMETRY_*`
- `REALITY_*`
- `DCC_*`
- `REPLAY_*`

## 규칙

- 새 issue code는 먼저 registry에 추가한다.
- 같은 의미의 issue code를 중복 생성하지 않는다.
- critical issue를 warn으로 숨기지 않는다.
