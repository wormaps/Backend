# Deterministic Replay

## 기준

동일 snapshot bundle, 동일 scope, 동일 schema/rule/package version은 manifest-compatible 출력을 만들어야 한다.

## Byte Hash 예외

환경 차이로 GLB byte hash가 달라질 수 있으면 다음 값은 같아야 한다.

- entity stableId set
- RenderIntent visualMode distribution
- MeshPlan budget summary
- QA issue code distribution
- final Reality Tier
